# Titanic EDA

## Overview
Exploratory Data Analysis on the Titanic dataset (891 passengers, 15 columns).
Analysed survival patterns across gender, passenger class, age, and fare.

## Key Findings
- Female passengers survived at 74% vs 19% for males
- First class survival rate (63%) was nearly 3x that of third class (24%)
- Fare is right-skewed — driven by passenger class, not age
- Pclass and fare are strongly negatively correlated (r = -0.55)
